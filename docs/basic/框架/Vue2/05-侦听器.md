# 侦听器 (`watch`)

## 含义

监听数据的变化，做出特定操作。本质是个函数，函数名为要监听的数据。

## 写法

在 vm 对象中定义 `watch: { ... }`

- 方法格式：操作简单，开发中首选。

  ![avatar](/images/watch方法格式.png)

      缺点 1：没办法在一开始就自动触发。
      缺点 2：如果侦听的是对象，对象属性发生变化是侦听不到的。

- 对象格式：

  ![avatar](/images/watch对象格式.png)

      优点 1：可以用 `immediate` 选项设置自动触发。
      优点 2：可以用 `deep` 选项设置是否侦听到对象属性的变化。

  - 注意：当侦听的对象是对象属性，侦听时要用单引号圈起来。
